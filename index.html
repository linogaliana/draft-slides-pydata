As data scientist

I needed a good playground

  • To experiment, data scientists need sandboxes and freedom
  • To simplify the transition from experimentation to production, data scientists need adapted tools
  • Many IT infrastructures are not adapted to data science
  • Having a reliable datalab has fostered the adoption of data science innovations

When I started to teach data science

Well, I needed the same

  • Website pythonds.linogaliana.fr/ is a good place to start, but…
  • … Notebooks more practical for learning
    • Code execution, text cells, etc.
    • Students need a sandbox area to discover ✏️
  • Notebooks supported by user-friendly environments: VSCode and Jupyter

Barriers to learning

  • Usual obstacles encountered in practical courses :
    • Students: waste time installing environments rather than practicing 😵‍💫.
    • Teachers: hard to manage as many installations as students 🤔
  • Desktop installations in educational institutions create as many problems as they solve:
    • Windows computers, limited rights to install packages, proxy…

Solution

Making notebooks available in a standardized environment via a web service:

  • Colab ? ❌
  • SSPCloud ✅

Why not turning ourselves to Colab?

  • Handy if you don’t have an environment, but…
  1. Commercial product:
    • 💸 Premium features for a fee (GPU)
    • Accustoms students to integrated services such as Drive.
  2. Limited freedom:
    • No direct access to a command line
    • No storage service adapted to data (Drive vs modern cloud technology)…

What about SSPCloud ?

  • 🚀 Both Colab and SSPCloud offer immediate startup
    • Runtime environment accessible from clickable link
  • 😍 Wide range of opensource technologies to go further
    • S3 storage, MLFlow, PostgreSQL, Argo workflow, etc.
    • Students can discover them during the rest of their data science course.
  • 💡 Philosophy :
    • Students discover IT principles with a well configured environment ;
    • But are not locked into a particular software they would need to purchase afterwards.
